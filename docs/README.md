## 🔍 Finding Documentation

### By Topic
- **Setup & Configuration** → Check `getting-started/` and individual module folders
- **Architecture** → Check `architecture/`
- **Specific Features** → Check module-specific folders (e.g., `image-compression/`)

### By Type
- **Quick Start** → Files in `getting-started/`
- **Technical Details** → Files in `architecture/`
- **Module Documentation** → Files in module-specific folders

## 📝 Adding New Documentation

When adding new documentation:
1. **Module-specific docs** → Place in the appropriate module folder (e.g., `docs/image-compression/`)
2. **General guides** → Place in `getting-started/`
3. **Architecture docs** → Place in `architecture/`
4. **Update this README** → Add links to new documentation
